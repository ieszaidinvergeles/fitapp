# Backend Documentation

This folder contains Laravel backend documentation for the Voltgym REST API.

## Contents

- `laravel_controllers.md`: controller behavior and API endpoints
- `laravel_api_v1_endpoints.md`: full v1 endpoint catalog (all declared API calls)
- `laravel_dashboards.md`: dashboard and reporting behavior
- `laravel_database_auth.md`: authentication-related database notes
- `laravel_database_logs_review.md`: logging schema review
- `laravel_database_review.md`: database review and fixes
- `laravel_database_validators.md`: validation strategy and FormRequests
- `laravel_demo_seeders.md`: seeded demo dataset and accounts
- `laravel_gym_logic.md`: gym-specific business rules
- `laravel_models.md`: Eloquent model notes
- `laravel_security_and_resources.md`: authorization and API resource notes
- `test/`: backend testing documentation
