# Frontend Documentation

This folder contains documentation for the WordPress frontend that consumes the FitApp Laravel API.

## Contents

- `wordpress_frontend_integration.md`: main integration guide for the WordPress team, including page-to-endpoint mapping
- `wp_phase_explanation.md`: frontend delivery scope and phase notes
- `test/`: frontend QA and testing documentation

## Notes

The implementation lives in `wordpress/wordpress-theme/` and the plugin work lives in `wordpress/wordpress-pluggin/`.
