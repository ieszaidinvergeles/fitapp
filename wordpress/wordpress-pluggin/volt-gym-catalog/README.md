# Volt GYM Catalog

Base skeleton for the WordPress plugin that registers the `Volt GYM` catalog as a Custom Post Type.

Current scope:
- Catalog-only content for products, supplements, and merchandise.
- No eCommerce checkout or payment flow.
- REST enabled so the client app can consume the catalog later.

Pending integration:
- Capability mapping if WordPress roles must mirror Laravel roles like `assistant`, `manager`, or `admin`.
- Custom meta fields for catalog price, SKU, stock status, featured badge, and gallery.
- Frontend templates inside the WordPress theme/client views.
